## プロジェクト情報

- **技術スタック**: Vue 3 + Vite + TypeScript + Phaser 3 + UnoCSS
- **パッケージマネージャー**: pnpm（npm使用禁止）
- **デプロイ先**: GitHub Pages

## 開発コマンド
```bash
# 開発サーバー起動（Windows）
run-dev.bat
# または
pnpm dev

# ビルド
pnpm build

# テスト（現在Windows環境で問題あり）
pnpm test
```

## 注意事項
- pnpmを必ず使用すること（npm使用禁止）
- エラーが発生したら根本原因を特定してから対処する
- プレイヤーファーストの姿勢を維持する
