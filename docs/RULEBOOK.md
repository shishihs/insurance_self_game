# ライフ・インシュアランス・サバイバル (Life Insurance Survival) ルールブック【改訂版】

## 1. ゲームの概要
**「人生は、攻めと守りのバランスだ。」**

プレイヤーは一人の人間として、青年期から充実期までの人生を歩みます。
夢を追いかけ、キャリアを築き、家族を持つ喜びを味わう一方で、病気や事故といった予期せぬリスクが常に隣り合わせにあります。
限られた「活力」を使い、いつ訪れるかわからないトラブルに「保険」で備えつつ、人生の目標（夢）を達成することを目指す、1人用デッキ構築型ライフシミュレーションゲームです。

**プレイ時間**: 30〜45分  
**対象年齢**: 14歳以上

## 4. ゲームの準備（セットアップ）

1. **夢の選択**: 夢カードをシャッフルし3枚引く。1枚を選んで「目標」エリアに表向きで置く。残りは箱に戻す。

2. **初期デッキ**: 初期人生カード18枚を受け取り、シャッフルして山札とする。

3. **活力セット**: 活力マーカーを「80」に置く。（難易度により変動）

4. **試練デッキの構築**:
   - 各ステージの試練カード12枚にトラブルカード3枚を混ぜる（計15枚）。
   - それぞれシャッフルして3つの山を作る。

5. **老化デッキ**: 老化カード10枚をシャッフルして山にする。

6. **保険市場**: 保険カード16枚をシャッフルし、4枚を表向きで並べる（保険市場）。残りは山札。

7. **最終試練**: 最終試練カード2枚からランダムに1枚選び、裏向きで脇に置く。

8. **獲得用カード**: 獲得用人生カード22枚を種類別に分けて並べる。

## 5. ゲームの進行

ゲームは複数の「ターン」で構成され、各ターンは以下のフェーズで進行します。

### フェーズ1: ドロー (Draw Phase)
- デッキから手札が上限（基本5枚）になるまでカードを引く。
- ※以前のルール（1枚ドロー）から変更され、常に十分な手札で戦略を立てられるようになりました。
- **山札が尽きた場合**:
  1. 老化カードを1枚引き、捨て札に加える。
  2. 捨て札をすべてシャッフルして新しい山札を作る。
  3. 老化デッキが尽きたら**即座に敗北**。

### フェーズ2: 試練の提示 (Challenge Phase)
- 現在のステージの試練デッキから**2枚**めくり、場に出す。
- 2枚のうち**1枚を選んで**挑戦する。選ばなかったカードは試練デッキの一番下に戻す。
- **トラブルカード**が出た場合:
  1. 即座にペナルティを受ける（保険で軽減可能）。
  2. トラブルカードを捨て、追加で1枚めくる。
  3. トラブルが2連続で出た場合、3枚目は引かずにターン終了。

### フェーズ3: アクション (Action Phase)

#### 3-A. カードのプレイ
- 手札から人生カードを好きな枚数出す。
- 各カードの**コスト**分の活力を支払う。
- カードの**パワー**を合計する。

#### 3-B. 追加ドロー（任意）
- **活力を5支払うごとに**、山札から1枚追加で引いてプレイできる。
- 試練カードに書かれた「ドロー枚数」を超えて引く場合に適用。

#### 3-C. 保険の適用
- 契約中の保険カードの効果を適用できる。

#### 3-D. 判定
**合計パワー ≧ 試練のパワー → 成功！**
- 試練カードに書かれた報酬を得る。
- 「保険の勧誘」: 保険市場から1枚を契約コストを払って獲得できる。

**合計パワー < 試練のパワー → 失敗...**
- **差分のダメージ**を受ける（活力を失う）。
- **救済**: 失った活力1点につき、プレイしたカード1枚をゲームから除外できる。
  - これにより弱いカードをデッキから取り除ける。
- 活力が0以下になったら**ゲームオーバー**（生命保険で1回だけ復活可能）。

### フェーズ4: 保険フェーズ (Insurance Phase)
#### 4-A. 保険の購入（任意）
- 保険市場から1枚を契約コストを払って獲得できる（1ターン1枚まで）。
- 獲得したら保険市場を1枚補充。

#### 4-B. 保険の維持
- 契約中の保険カードの維持コストを支払う。
- 支払わない場合は解約（捨て札ではなく保険市場の山に戻す）。

#### 4-C. 定期保険の確認
- 定期保険は契約から10ターン経過で失効。
- ステージ移行時、定期保険の維持コスト+2。

### フェーズ5: ターン終了 (End Phase)
- プレイした手札・使わなかった手札をすべて捨て札にする。
- ターンマーカーを1つ進める。
- **試練デッキが尽きたら**次のステージへ移行。

## 6. ステージと進行

### 青年期 (Youth) - 15枚の試練デッキ
- 開始活力: 80
- 手札上限: 5枚
- 特徴: 試練は比較的簡単。保険料が安い時期。
- 目標: デッキの基盤を作る。

### 中年期 (Middle Age) - 15枚の試練デッキ
- 手札上限: 5枚
- 特徴: 試練の難易度上昇。定期保険の維持コスト+2。
- 追加ルール: 「家庭」「仕事」のコンボが重要になる。

### 充実期 (Fulfillment) - 15枚の試練デッキ
- **活力上限が70に低下**（80以上あっても70に減少）。
- 手札上限: 5枚
- 特徴: 試練の難易度最大。老化カードの影響が深刻化。
- 追加ルール: ターン開始時に活力-2（老化による消耗）。

### 最終試練
- 充実期の試練デッキが尽きたら、最終試練カードを公開。
- 最終試練に勝利すればゲームクリア。

---

## 7. 勝利条件と敗北条件

### 勝利
- 全3ステージを生き抜き、**最終試練に勝利**する。

### 敗北
以下のいずれかで敗北:
- 活力が0以下になる（生命保険未契約、または使用済み）。
- 老化デッキが尽きた状態で山札を作り直す必要が生じる。

### 夢の達成
- ゲームクリア時に夢カードの条件を満たしていれば、ボーナス点を獲得。
- 条件を満たしていなくてもクリアは可能（ボーナスなし）。

## 8. スコア計算

ゲーム終了時（勝敗問わず）にスコアを計算します。

| 項目 | 点数 |
|:---|:---|
| 残り活力 | 1点につき+1点 |
| デッキ内の人生カードの合計パワー | そのまま加算 |
| 老化カード | 1枚につき-5点 |
| 未消化の試練カード | 1枚につき-3点 |
| 最終試練クリア | +30点 |
| 夢の達成 | 夢カードに記載のボーナス点 |
| 契約中の永続保険 | 1枚につき+5点 |

**ランク目安**:
- 100点以上: S（完璧な人生）
- 80〜99点: A（充実した人生）
- 60〜79点: B（まずまずの人生）
- 40〜59点: C（苦労の多い人生）
- 39点以下: D（波乱万丈）

- 老化カードから「孤独」を除外。
- トラブルカードを各ステージ2枚に減らす。

- 初期活力: 80

### レベル3: 上級

- 充実期の活力上限が60に低下。

- 初期活力: 60
- ゲーム開始時に老化カード2枚を山札に混ぜる。
- 充実期の活力上限が60に低下。

## 11. FAQ・特殊状況

**Q: 山札も捨て札も0枚の場合は？**  
A: 老化カードを1枚引き、それだけで新しい山札を作ります。

**Q: 保険市場が尽きたら？**  
A: 捨てられた保険カードをシャッフルして新しい山にします。

**Q: 活力がマイナスになったら？**  
A: 生命保険がなければ即敗北。生命保険があれば発動し、記載の活力で復活。

**Q: 試練を1枚も出さずにパスできる？**  
A: いいえ。必ず試練に挑む必要があります。ただし手札を0枚でプレイする（=確実に失敗する）ことは可能です。

**Q: 同じ保険を複数契約できる？**  
A: いいえ。同名の保険は1枚まで。

## 12. プレイ例（1ターンの流れ）

**状況**: 青年期3ターン目、活力72、手札上限5枚

1. **ドロー**: 山札から5枚引く→「ジョギング」「勉強」「アルバイト」「食事」「読書」

2. **試練の提示**: 試練デッキから2枚めくる→「就職活動(パワー6)」と「資格試験(パワー7)」。「就職活動」を選択。

3. **アクション**: 
   - 「ジョギング」(コスト0/パワー1)をプレイ
   - 「勉強」(コスト5/パワー2)をプレイ→活力67
   - 「アルバイト」(コスト5/パワー2)をプレイ→活力62
   - 合計パワー: 5
   - パワー不足なので活力5を払って追加ドロー→「休息」(パワー0)...まだ足りない
   - さらに活力5を払って追加ドロー→「散歩」(パワー1)
   - 合計パワー: 6 = 試練パワー6 → **成功！**
   - 報酬: 「正社員」カードを獲得してデッキに加える

4. **保険フェーズ**: 保険市場に「医療保険（定期）」がある。契約コスト5を払って契約→活力52

5. **ターン終了**: プレイしたカード・残った手札を捨て札に。ターンマーカーを進める。
