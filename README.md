# 人生充実ゲーム (Life Fulfillment Game)

生命保険を「人生の味方」として描く、一人用デッキ構築ゲーム

## 🎮 ゲーム概要

プレイヤーは人生の各ステージで様々なチャレンジに立ち向かい、保険カードを活用しながら充実した人生を送ることを目指します。

## 🎯 操作方法

### マウス操作
1. **ゲームをプレイ**ボタンをクリックしてゲーム開始
2. カードをクリックで選択/選択解除
3. カードをドラッグしてチャレンジエリアにドロップ
4. アクションボタンで各種操作

### キーボード操作
- **Tab**: 次の要素へフォーカス移動
- **矢印キー**: 方向指定のナビゲーション
- **Space/Enter**: 選択/決定
- **Esc**: キャンセル/選択解除
- **D**: カードをドロー
- **C**: チャレンジ開始
- **E**: ターン終了
- **1-7**: 手札カードを直接選択
- **M**: サウンドのON/OFF切り替え
